
# AI-Augmented Framework for Personalized Student Timetabling

This repository contains the LaTeX source code and build system for the MO903-T3 final coursework. It features a modular structure, automated acronym management, and an optimized build pipeline for Arch Linux.

---

## 🚀 Build and Live Preview

To compile the document with a live-reloading preview in **Zathura**, run the following command. This handles the output directory, sets the necessary environment paths for BibTeX, and ensures Zathura closes when you stop the process (Ctrl+C).

```bash
mkdir -p output && find output -type f -delete && \
trap 'pkill -P $$ zathura' EXIT; \
BIBINPUTS="./src/sections/:" \
BSTINPUTS="./src/sections/:" \
TEXINPUTS="./src//:./src/img//:" \
latexmk -pdf -pvc -e '$pdf_previewer="zathura %S"' -outdir=output src/main.tex
```

## ✍️ Key Features

### 1. Smart Acronym Management
Acronyms are defined globally in `src/structure/engine.tex`. 
* **Automatic Expansion:** The first time an acronym command (e.g., `\PSTP`) is used, it renders the full name. All subsequent uses render only the abbreviation.
* **Intelligent Spacing:** Uses `\xspace` to ensure spaces are only added when necessary (preventing text from gluing to punctuation).

### 2. Reference Ordering
References are indexed in **reading order** (sequential) using the `unsrt` style, ensuring the bibliography matches the flow of the document.

### 3. Clean Sectioning
A custom patch in `src/sections/references.tex` removes the redundant internal header generated by the bibliography environment, ensuring "Section 5 References" renders correctly without duplicates.
